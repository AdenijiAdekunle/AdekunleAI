About

Hi! I'm Adekunle Adeniji, a Google-certified Data Analyst. I leverage my strong analytical skills to extract meaningful insights from complex datasets, bridging the gap between language and numbers. With a background in English and Literary Studies, I bring a unique perspective to data analysis, effectively communicating data-driven insights to various audiences. My meticulous attention to detail and problem-solving abilities enable me to unravel intricate data puzzles and empower organizations to make informed decisions. My journey from literature to data analysis underscores my adaptability and commitment to lifelong learning, as I strive to make impactful contributions in my field.

  • Programming: Python, SQL (MS SQL Server, BigQuery), HTML, CSS, JavaScript
  • Visualization: Excel, Tableau, Power Bi, Seaborn, Matplotlib
  • Other Tools: Numpy, Pandas, Microsoft Office Suite, NLTK

Projects

OLYMPICS HISTORY ANALYSIS
I performed data cleaning in Excel to ensure the dataset was complete and reliable, then used Power Query to transform and validate the data. Finally, I developed a 3-page Power BI report showcasing key historical insights through clear and interactive visualizations.
Documentation         Visualization         Github

DATA ANALYST JOBS ANALYSIS
I utilized Python to analyze data analyst job postings and uncover key insights on what it takes to secure a role in the field. Using the BeautifulSoup library, I scraped and gathered authentic job data, then performed data cleaning and exploration with SQL queries on Microsoft SQL Server. To highlight the required skill sets across different experience levels, I created a WordCloud visualization that effectively summarizes the most in-demand competencies.
Documentation         Github

DATA EXPLORATION IN SQL
